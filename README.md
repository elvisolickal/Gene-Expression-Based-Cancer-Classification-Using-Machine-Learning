# Gene-Expression-Based-Cancer-Classification-Using-Machine-Learning

Preprocessing of high-dimensional gene expression data.
Selection of the most informative genes using feature selection techniques.
Training and evaluation of multiple machine learning classifiers.
Comparison of models using metrics such as Accuracy, Precision, Recall, F1-score, and Confusion Matrix.
Identification of the best-performing model for classifying five cancer types.
Interpretation of model predictions using feature importance or explainability methods (e.g., SHAP or permutation importance).
Development of an interpretable system capable of predicting cancer type from gene expression profiles.
